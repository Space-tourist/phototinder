PHOTOTINDER

Приложение, позволяющее быстро отсортировать фотографии на понравившиеся и непонравившиеся. 
Для использования необходимо выбрать каталог с фотографиями, в котором автоматически будут созданы каталоги для понравившихся фотографий и непонравившихся.
Управление: Стрелка влево - dislike; стрелка вправо - like; пробел - skip(отложить), либо кнопки в интерфейсе.
В соответствии с выбором фотографии будут перемещены в созданные каталоги.
Поддерживаемые форматы: .JPG
Фотографам с 1000000 фото на карте памяти посвящается..:)
